# Adriano Gonzalez | Portfolio

Personal portfolio website built with HTML, CSS and JavaScript.

## Pages

- **index.html** — Main page: about me, projects and contact form
- **mascotas.html** — Secondary page: pet gallery

## Features

- Responsive design (mobile, tablet, desktop)
- Show/hide about me section with JavaScript
- Welcome message on page load
- Project cards with hover effects
- Contact form in the footer

## File Structure

```
portafolio/
├── index.html
├── mascotas.html
├── minified.css
├── fonts.css
├── script.js
├── Montserrat/
├── profile_img/
├── icons/
└── pets/
```

## How to Run

No installation needed. Just open `index.html` in any browser.
